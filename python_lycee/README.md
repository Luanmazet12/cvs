# Python Lycée — Exercices progressifs

Bienvenue ! Ce dossier contient 6 fichiers d'exercices Python pour apprendre à coder au niveau lycée.

## 📂 Organisation des fichiers

| Fichier | Thème | Niveau |
|---------|-------|--------|
| `01_variables.py` | Variables, types, print, input | ⭐ |
| `02_conditions.py` | if / elif / else | ⭐⭐ |
| `03_boucles.py` | Boucles for et while | ⭐⭐ |
| `04_listes.py` | Listes | ⭐⭐ |
| `05_fonctions.py` | Fonctions | ⭐⭐⭐ |
| `06_projets.py` | Mini-projets complets | ⭐⭐⭐ |

## 🚀 Comment utiliser ces fichiers

1. Ouvre un fichier dans ton éditeur (Thonny, VS Code, etc.)
2. Lis le cours rapide en haut du fichier
3. Essaie de faire l'exercice **sans regarder la correction**
4. Une fois que tu as essayé, décommente la correction pour comparer

> **Astuce** : pour décommenter une ligne, enlève le `#` au début.

## 🎮 Mini-projets

Lance directement `06_projets.py` pour jouer avec :
- 🎲 Jeu du nombre mystère
- 🔢 Calculatrice interactive
- 📊 Gestion de notes de classe
- 🪢 Le pendu

```bash
python 06_projets.py
```
